Shibboleth Identity Provider Installation

The following commands should be issued to install the Shibboleth Identity 
Provider.  You will be prompted for configuration information, including the 
desired name of the IdP web aplication and the locations in which the IdP 
components should be installed.

$ export JAVA_HOME=(the path to your java installation)
$ cd shibboleth-idp-install
$ ./ant install

From this point forward, refer to the deploy guide for configuration details.
